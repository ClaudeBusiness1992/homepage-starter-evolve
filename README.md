# homepage-starter

Basic homepage template für multiple use — pixel&code.

## Projekt

Moderne, responsive One-Page-Website für kleine Unternehmen. Vanilla HTML/CSS/JS, keine Dependencies.

## Sections

- **Hero** — Headline, Subtext, CTA
- **Über uns** — Team-Vorstellung mit Fotos
- **Leistungen** — 6 Service-Cards
- **Stats** — Kennzahlen-Leiste
- **Preise** — 3 Pakete (Starter / Professional / Premium)
- **Kontakt** — E-Mail & Telefon CTA
- **Footer**

## Setup

Keine Installation nötig — `index.html` direkt im Browser öffnen.

## Anpassen

Siehe `docs/customization.md` für Schritt-für-Schritt-Anleitung.  
Farben und Fonts: `config/theme.tokens.example`  
Inhalte: `config/client.config.example`

## Branches

| Branch | Zuständigkeit |
|---|---|
| `main` | Stable baseline |
| `feat/hero-about` | Nav, Hero, About |
| `feat/services-pricing` | Services, Stats, Pricing, Contact, Footer |
